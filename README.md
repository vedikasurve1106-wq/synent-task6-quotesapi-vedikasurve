# 🌅 Sunset Quotes – API Integration Project

A responsive web application that fetches and displays inspirational quotes using a public Quotes API. Built using **HTML, CSS, and JavaScript**, this project demonstrates API integration, asynchronous JavaScript, and dynamic content rendering.

## 🚀 Features

* Fetches a random inspirational quote from a public API
* Displays the quote along with the author's name
* Loading message while fetching data
* Error handling for failed API requests
* Copy quote to clipboard functionality
* Smooth animations and responsive design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Fetch API)
* DummyJSON Quotes API

## ▶️ How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Click **New Inspiration** to fetch a new quote.
